# Analyze optical properties
The aim of this session is to solve the Time Dependent Equation for the Reduced One Body Density Matrix

$
\begin{aligned}
\mathrm{i} \hbar \partial_t \underline{\rho}_{\mathbf{k}}(t)=\left[\underline{h}_{\mathbf{k}}^{\mathrm{rt}}[\underline{\rho}]+\underline{U}_{\mathbf{k}}^{\mathrm{ext}}(t), \underline{\rho}_{\mathbf{k}}(t)\right]-\mathrm{i} \Gamma_{\mathbf{k}} \underline{\rho}_{\mathbf{k}}
\end{aligned}
$

$
\begin{aligned}
\underline{h}_{\mathbf{k}}^{r t}[\underline{\rho}]=\underline{h}_{\mathbf{k}}^{e q}+\Delta \underline{h}_{\mathbf{k}}^{q p}+\Delta \underline{\Sigma}^{H x c}[\underline{\rho}]
\end{aligned}
$

$
\begin{aligned}
\Delta \underline{\Sigma}^{\mathrm{Hxc}}[\underline{\rho}]=\underline{\underline{K}}^{\mathrm{Hxc}} \cdot \underline{\rho}
\end{aligned}
$


1. First step is to convert the QE output to yambo running on the terminal
```bash
p2y
```
The initialize the yambo database with:
```yambo -F setup.in```
where `setup.in` contains the simple runlevel:

<details>
  <summary><strong>▶️ Expand for `setup.in` `yambo` input file</strong></summary>
setup
    <pre><code class="language-bash">
setup                        # [R INI] Initialization
    </code></pre>
</details>

K is the functional derivative of the self--energy with respect to the density matrix and corresponds to the kernel of the Bethe--Salpeter equation. In the language of the yambo code we will refer to it with the name "real time collisions" or simply **collisions**.


The collision file for the Fock potential can be generated by:
<details>
  <summary><strong>▶️ Expand for `collisions_Fonly.in` `yambo` input file</strong></summary>
    <pre><code class="language-bash">
collisions                   # [R] Eval the extended Collisions
HXC_Potential="FOCK"          # [SC] SC Potential
% COLLBands
  7 | 10 |                   # [COLL] Bands
%
EXXRLvcs= 107       RL    # [XX] Exchange RL components
    </code></pre>
</details>

and then run:
```bash
yambo_nl -F collisions_Fonly.in
```
Optionally, you can change the scattering potential to Hartree ``HARTREE`` instead of Fock ``FOCK``.
Other available options are ``HARTREE+FOCK``, ``HARTREE+SEX``. Note you might want to change potential and HXC_Potential.

```bash
# OLD
Potential= "HARTREE-FOCK"          # [SC] SC Potential
# NEW
HXC_Potential= "HARTREE+FOCK"          # [SC] SC Potential
```
You can also force computing Fxc via libxc with run-level `FxcLibxc`.
For now we focus on TD-HSEX computing only conduction and valence bands corrections.
